# Ifolia Portfolio

Public portfolio case study for **Ifolia Community Management System**.

This repository is intentionally separated from the private production source code. It contains only a static case study page and safe visual assets for HR/recruiter review.

## What Is Ifolia?

Ifolia is a web and PWA-based community management system for residential clusters. It helps residents and administrators manage daily operations such as:

- Resident and house-unit data
- IPL billing and payment monitoring
- Community finance reporting
- Digital letter request and approval workflow
- PDF/JPG document export with QR verification
- Security guard scheduling
- Complaints, announcements, agenda, and Telegram notifications

## Technical Highlights

- Full-stack product delivery
- Role-based access control
- Digital approval workflow
- Mobile-first PWA improvements
- Document rendering and export flow
- Backend, frontend, database, and deployment awareness

## Privacy Note

No real resident data, credentials, environment variables, or internal production source code are included in this public portfolio repository.
